Hospital Manager

  • Category: Web App
  • Client: Make Believe Hospital
  • Project date: 8 August, 2023
  • Github: https://github.com/marknosal/hospital-manager

CLI used to perform full CRUD on Doctors, Patients, and Appoiintments.

This was the first project I made using Python. There is no front end here. It's strictly a CLI that a hypothetical hospital could use to manager their doctor/patient/appointment records.

  • Utilize Python CLI to perform full CRUD on all records
  • Clean CLI allows effecient management of hospital records
  • Python logic will check and prevent duplicate dataq entries
  • SQLite is the database I used
